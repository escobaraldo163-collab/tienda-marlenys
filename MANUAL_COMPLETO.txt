IMPORTANTE
Esta versión NO se abre con doble clic. Es un proyecto profesional y necesita Node.js.

PASO 1 - INSTALAR NODE.JS EN WINDOWS 11
1. Entra a https://nodejs.org
2. Descarga la versión LTS.
3. Instálala.
4. Reinicia la computadora si Windows lo pide.

PASO 2 - ABRIR EL PROYECTO
1. Descomprime este ZIP.
2. Entra a la carpeta tienda-marlenys-plataforma.
3. En la barra de dirección escribe:
   cmd
4. Presiona Enter.

PASO 3 - INSTALAR
En la ventana negra escribe:

npm install

PASO 4 - PROBAR
Luego escribe:

npm run dev

Abre en Google Chrome:

http://localhost:3000

PANEL ADMINISTRATIVO
El panel ya permite agregar, editar y eliminar productos en modo local.
Los cambios se guardan en esa computadora usando localStorage.

SIGUIENTE FASE
Conectar Supabase para que los cambios se guarden en internet y puedas administrar desde cualquier computadora.
